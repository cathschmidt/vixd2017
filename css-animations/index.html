There are two ways to animate using CSS—

Transitions

Hover On Me !

↓

          
div.transition-example {
  font-size: .5em;
  background-color: rgb(20,100,255);
  padding: 5vw;
  margin: 2vw;
  height: 5vw;
  width: 5vw;
  transition: all 2s;
}
/* this div has properties that change on hover */
div.transition-example:hover {
  background-color: rgb(255,100,20);
  transform: rotate(360deg);
}

Transitions require an interaction like:

  • hovering over an element
  • changing the width of the browser

Animation

I just do my thing ;)

↓

          
div.animate-example {
  font-size: .5em;
  background-color: rgb(20,100,255);
  padding: 5vw;
  margin: 2vw;
  height: 5vw;
  width: 5vw;
  /* this is the good stuff! */
  animation-duration: 4s;
  animation-name: spin;
  animation-iteration-count: infinite;
}
/* here's the info that describes our animation */
@keyframes spin {
  50% {
    background-color: rgb(255,100,20);
    transform: rotate(360deg);
  }
  100% {
    background-color: rgb(20,100,255);
    transform: rotate(-360deg);
  }
}

The CSS Animation property doesn't require an interaction and is more detailed.

Animation properties include.

  • animation-duration — how long is the animation
  • animation-iteration-count — does it loop?
  • animation-timing-function — the pace of the animation; this is how we get into easing!
  • and many more...
